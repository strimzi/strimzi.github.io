Welcome to the Strimzi documentation! This page provides comprehensive guides and resources to help you deploy, manage, and optimize Kafka clusters using Strimzi. Whether you're new to Strimzi or an experienced user, you'll find everything you need to get started.

Strimzi simplifies Kafka cluster management through a set of specialized operators:

  • Cluster Operator handles the lifecycle of Kafka clusters, ensuring smooth deployment and management. If you're running the Cluster Operator, you can also use the Drain Cleaner to assist with pod evictions.
  • Topic Operator manages Kafka topics, making it easy to create, update, and delete topics.
  • User Operator manages Kafka users, including authentication credentials.

Additionally, Strimzi's Kafka Bridge provides a RESTful interface for producing and consuming messages, making it easier to integrate Kafka with your applications.

Explore the guides below, or jump directly to the Overview for a comprehensive introduction to Strimzi operators and features.
